Momonga MQTT Documentation
==========================

About Momonga
`````````````

Welcome to the Momonga documentation!

Momonga is an MQTT Server. It is supported MQTT3.1.1 specification, and written in Golang.
You can easy to deploy Momonga MQTT Server in any environment.

Momonga goals are provide durable middleware in Go, extendable MQTT server and fast delivery.
